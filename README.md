# Industic - Factory & Manufacturing React Website

Multi-page React (Vite) website matching the Industic theme design.

## Pages
- `/` Home
- `/company` Company
- `/industries` Industries (list)
- `/industries/:slug` Industry detail (6 sub-industries)
- `/news` News & Media
- `/contacts` Contacts

## Run
```
npm install
npm run dev
```

## Build
```
npm run build
```
